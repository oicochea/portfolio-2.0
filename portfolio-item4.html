Garage Vision

Virtual Car Garage

Live App Github

Technologies Used:
Rails,JSON,API,One to Many Relationship,Scaffold,CRUD,Models,Schema,Heroku, Netlify

A webApp that is a virtual Garage. Allows creating user and JWT token auth. Users can create vehicles, delete a vehicle and edit a vehicle. Each vehicle has a Make, Model,Year, Miles, Image and Services Section. The this can be editable , the user is recommended to keep adding services and the date to maintain a log of the services the car has received
